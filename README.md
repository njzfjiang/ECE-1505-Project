## Comparing First-Order Methods for L1-Regularized Least Squares
ECE 1505 project

### How to Run
0. Make sure python is installed, create a python virtual environment if needed

1. Install all requirements in requirements.txt using ```pip install -r requirements.txt``` at the project's root folder

2. Run python experiments.py --all or --rq1, --rq2, --rq3 to generate data for the experiments

3. Run plots.py to generate plots for the data(make sure all experiments has been ran)

4. For real data experiment, run ```python experiments_real.py``` to create the data and create the plots
